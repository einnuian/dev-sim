# React Emergency-Meeting Standup Demo

Basic animated standup simulation using React (CDN) and no build tooling.

## Run

- Open `index.html` directly in a browser, or
- serve the folder with any static server.

Example:

```bash
cd /home/mayira/dev-sim/examples/react-amongus-standup-demo
python3 -m http.server 8080
```

Then visit: `http://localhost:8080`

## What it demonstrates

- Turn-taking standup rhythm
- Distinct role speakers (SM, FE, BE, TL, SA)
- Emergency-meeting-inspired visual style (portrait cards + chat bubbles)
- No direct character likenesses and no "sus" references
- Animated active speaker + running conversation log
